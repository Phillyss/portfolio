AR Library

A fully functioning Augmented Reality web application I designed and developed for the HvA Robot Lab. Solo internship project.

The Robot Lab needed a convenient place to store the 3D assets they produce with their students. Using phones, these 3D assets should be showcasable in AR.

Only accessible to HvA staff and students. Users are able to upload 3D files, name them, tag them, etc. Uploads can be filtered through tags like materials or projects. Detail pages show additional information and a button to launch an AR view.

GameBuddy

A matching web application designed to make new friends with similar gaming interests. Create an account, select your favourite games, find and favourite other people!

My first backend web development project. Runs on an Express server with EJS templating and a MongoDB database.

The backend part I mainly developed was the favourite feature and making sure these people don't show up in your feed again. I also wrote image compression for profile pictures.
